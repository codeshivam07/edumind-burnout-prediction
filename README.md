# EduMind Burnout Prediction System

EduMind is an AI-powered system that predicts student burnout and academic performance using machine learning models.

## Tech Stack

### Backend
- Node.js
- Express.js

### ML Service
- FastAPI
- Scikit-learn

### Database
- MongoDB

### ML Models
- Burnout Prediction Model
- Student Clustering Model
- Academic Performance Prediction Model

---

# Project Architecture

Frontend (React)
↓
Node.js Backend (API + Auth)
↓
FastAPI ML Service
↓
Machine Learning Models
↓
MongoDB Database
